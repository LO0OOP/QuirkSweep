# 扫雷游戏

一个功能丰富的扫雷游戏，支持经典模式和趣味闯关模式，包含云端排行榜、特殊道具系统等多种特色玩法。

## 🎮 游戏模式

### 经典模式
传统扫雷玩法，支持多种自定义设置：

- **难度预设**：简单（9×9，10雷）、中等（16×16，40雷）、困难（16×30，99雷）
- **自定义设置**：可自由调整行数、列数和地雷数量
- **无猜模式**：确保所有谜题都可通过逻辑推理解决，无需猜测
- **历史记录**：本地保存最近50局游戏记录
- **云端排行榜**：标准难度支持上传成绩，与全球玩家竞争前50名

### 趣味闯关模式
创新玩法，加入Roguelike元素：

- **生命系统**：初始生命值，踩雷扣除生命而非直接结束
- **倒计时挑战**：在限定时间内完成挑战
- **特殊格子**：
  - ❤️ **生命格**：生命值+1（上限5点）
  - ⏰ **时间格**：50%概率+15秒，50%概率-10秒
  - ⚡ **雷电格-黄**：直接触发，十字形揭开并自动标记地雷
  - ⚡ **雷电格-红**：连锁揭开上下左右四格
  - 🔍 **扫雷器**：直接触发，自动标记周围8格中的地雷
  - 🌫️ **迷雾格**：5×5范围被迷雾遮盖15秒

## 🛠️ 技术栈

- **前端框架**：React 18 + Vite
- **样式方案**：Tailwind CSS
- **UI组件**：shadcn/ui
- **状态管理**：React Hooks + Context
- **数据存储**：Supabase（云端排行榜）+ LocalStorage（本地备份）
- **图标库**：Lucide React
- **路由**：React Router（HashRouter）

## ✨ 核心功能

### 无猜模式算法
使用约束传播和回溯算法生成保证可解的棋盘，玩家可以通过纯逻辑推理完成游戏，无需猜测。

### 排行榜系统
- 支持云端存储（Supabase）和本地存储双模式
- 当云端服务不可用时自动降级到本地存储
- 显示前50名成绩，支持按难度和模式筛选
- 自动生成虚假排行榜数据供体验

### 特殊格子系统
闯关模式引入6种特殊格子，增加游戏随机性和策略性，每种格子都有独特的视觉效果和交互反馈。

## 📝 游戏规则

### 经典模式
- 左键揭开格子，右键标记/取消标记地雷
- 数字表示周围8格中的地雷数量
- 揭开所有非雷格子即可获胜
- 踩到地雷游戏结束

### 闯关模式
- 在限定时间内清除所有地雷即可通关
- 踩到地雷扣除1点生命值，生命值为0时失败
- 合理利用特殊格子可以事半功倍
- 迷雾覆盖区域无法操作
